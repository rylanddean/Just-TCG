Frequently Asked Questions

Just TCG loads its card catalog from Limitless TCG. If the catalog looks empty or stale, go to Settings → Data → Force network sync to trigger a fresh download. This requires an internet connection. The sync takes 10–30 seconds depending on your connection speed.

If your device has never had an internet connection after installing the app, the bundled seed data will be used as a starting point. Open any internet connection and sync to get the full current catalog.

The import expects the standard PTCGL copy-paste format. Make sure you copy the full list from PTCGL (Deck Details → Export → Copy to Clipboard). The list should start with Pokémon: X and include quantity, name, set code, and collector number for each card.

If a card can't be matched automatically, an unmatched card swap sheet will appear so you can pick the correct version manually. Basic Energy cards sometimes use non-standard formatting — the swap sheet handles those too.

For best results: hold the card steady, ensure the set symbol (bottom left of the card) and collector number (bottom right) are both fully visible and in focus. Good lighting without glare makes a significant difference.

The scanner reads the collector number and set code printed at the bottom of the card — not the card illustration or name. Holographic foil cards sometimes cause glare; try tilting the card slightly to reduce reflection.

AI features require iOS 18.1 or later and a device with an A17 Pro chip or later (iPhone 15 Pro, iPhone 16 family, and newer). On devices that don't meet these requirements, the AI features are hidden.

If your device meets the requirements but features still don't appear, ensure that Apple Intelligence is enabled in Settings → Apple Intelligence & Siri.

Tournament data is fetched live from Limitless TCG when you open the Tournaments tab, then cached for up to 1 hour. If you're seeing older data, pull down on the tournament list to force a refresh.

If the refresh fails, an error banner will appear. Check your internet connection. If Limitless TCG is experiencing downtime, the cached data will be shown until the service recovers. Just TCG is not affiliated with Limitless TCG.

Match history syncs across devices via iCloud. Check the following:

  • Both devices are signed into the same Apple ID
  • iCloud is enabled for Just TCG on both devices (Settings → [Your Name] → iCloud)
  • The new device has had a few minutes to sync after first launch

If sync doesn't recover your data, use a backup exported from Settings → Export Backup on your original device, then import it on the new device via Settings → Import Backup.

Go to Settings → Data → Export Backup. This creates a JSON file containing all your decks and match history. Share it to Files, AirDrop, or any other destination using the iOS Share Sheet.

To restore, go to Settings → Data → Import Backup and select the file. Decks that already exist on the device will be skipped to prevent duplicates.

Card sectioning is determined by the card's subtype in the local catalog. If a card appears in the wrong section, it may have incomplete or incorrect catalog data. Try forcing a card cache sync via Settings → Developer → Force network sync (visible in the release build as Force network sync under Data).

If the problem persists after syncing, please open a GitHub issue with the card's name, set code, and collector number so we can investigate the catalog entry.

Yes. Deck building and match logging work completely offline using the locally cached card data. Analytics are also computed entirely from your local match records.

Features that require a connection: card catalog refresh, tournament feed, and fetching new deck list data. These will show cached data when offline and refresh when connectivity returns.

Just TCG supports the Standard format only. The card catalog, deck legality validation, and tournament data are all specific to Standard. Expanded and Pocket formats are not currently supported.
